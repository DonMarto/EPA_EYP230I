Se construye un archivo de Jupyter Notebook para una exploración inicial de los datos (combine_data.ipynb), y se generan diferentes estructuras de los datos (dataframes.pkl, dataframes.xlsx), a partir de los datos originales por año (data.zip).

Sin embargo, para el análisis posterior se hace uso únicamente de los archivos vehicles.csv y vehicles.xlsx, que corresponden a los datos compilados por la misma organización EPA.

Todos los datos son extraídos en la página oficial del proyecto Fuel Economy (fueleconomy.gov) del departamento de energía de los estados unidos. Esta organización reserva los derechos de estos datos, pero permite su libre uso para actividades académicas, como el presente proyecto, desligándose de las conclusiones que podamos obtener.

Link de descarga: https://www.fueleconomy.gov/feg/download.shtml
Link de disclaimer: https://www.fueleconomy.gov/feg/ws/index.shtml#vehicle
